Módulo de interface para o programa DESSEM
============================================

O *idessem* é um pacote Python para manipulação dos arquivos
de entrada e saída do programa `DESSEM <https://www.cepel.br/linhas-de-pesquisa/dessem/>`_. O DESSEM é
desenvolvido pelo `CEPEL <http://www.cepel.br/>`_ e utilizado para a programação da operação do Sistema Interligado Nacional (SIN).

O idessem oferece:

- Meios para leitura dos arquivos de entrada e saída do DESSEM
- Facilidades para estudo e análise dos dados utilizando DataFrames de `pandas <https://pandas.pydata.org/pandas-docs/stable/index.html>`_
- Dados estruturados em modelos com o uso do paradigma de orientação a objetos (OOP)
